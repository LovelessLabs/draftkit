# Filter Utilities (Tailwind CSS v3)

## Blur
```
blur-none → filter: blur(0)
blur-sm   → filter: blur(4px)
blur      → filter: blur(8px)
blur-md   → filter: blur(12px)
blur-lg   → filter: blur(16px)
blur-xl   → filter: blur(24px)
blur-2xl  → filter: blur(40px)
blur-3xl  → filter: blur(64px)
```


## Common Patterns

### Blurred Background
```html
<div class="backdrop-blur-md bg-white/30">
  Frosted glass effect
</div>
```

### Image Hover Effect
```html
<img class="grayscale hover:grayscale-0 transition-all" src="photo.jpg" alt="">
```

### Disabled Image
```html
<img class="grayscale opacity-50" src="photo.jpg" alt="">
```

### Vintage Photo Effect
```html
<img class="sepia contrast-125" src="photo.jpg" alt="">
```

### Brightened on Hover
```html
<img class="brightness-90 hover:brightness-110 transition-all" src="photo.jpg" alt="">
```

### Drop Shadow on PNG
```html
<img class="drop-shadow-lg" src="logo.png" alt="">
```

### Modal Backdrop
```html
<div class="fixed inset-0 backdrop-blur-sm bg-black/25">
  <div class="modal">Content</div>
</div>
```

### Navigation Glass Effect
```html
<nav class="backdrop-blur-lg bg-white/70 sticky top-0">
  Navigation
</nav>
```

### Combining Filters
```html
<img class="blur-sm brightness-75 contrast-125 saturate-150" src="photo.jpg" alt="">
```

### Dark Mode Invert
```html
<img class="dark:invert" src="logo.svg" alt="">
```
