## Common Patterns

### Basic 3-Column Grid
```html
<div class="grid grid-cols-3 gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

### Responsive Grid
```html
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-6">
  <!-- 1 col mobile, 2 cols tablet, 4 cols desktop -->
</div>
```

### Dashboard Layout (12-Column)
```html
<div class="grid grid-cols-12 gap-4">
  <aside class="col-span-3">Sidebar</aside>
  <main class="col-span-9">Main content</main>
</div>
```

### Card Grid with Feature Card
```html
<div class="grid grid-cols-3 gap-4">
  <div class="col-span-2 row-span-2">Featured</div>
  <div>Card 1</div>
  <div>Card 2</div>
</div>
```

### Auto-fit Responsive Grid
```html
<div class="grid grid-cols-[repeat(auto-fit,minmax(250px,1fr))] gap-4">
  <!-- Auto-responsive without breakpoints -->
</div>
```

### Subgrid Layout (v4)
```html
<div class="grid grid-cols-3 gap-4">
  <div class="col-span-3 grid grid-cols-subgrid">
    <!-- Inherits parent's column sizing -->
    <div>Aligned to parent column 1</div>
    <div>Aligned to parent column 2</div>
    <div>Aligned to parent column 3</div>
  </div>
</div>
```

### Holy Grail Layout
```html
<div class="grid grid-rows-[auto_1fr_auto] min-h-screen">
  <header>Header</header>
  <main class="grid grid-cols-[200px_1fr_200px]">
    <aside>Left sidebar</aside>
    <article>Content</article>
    <aside>Right sidebar</aside>
  </main>
  <footer>Footer</footer>
</div>
```

### Centered Grid Items
```html
<div class="grid place-items-center min-h-screen">
  <div>Centered content</div>
</div>
```

## v3 to v4 Migration Notes

| Change | v3 | v4 |
|--------|----|----|
| Gap values | Static (`gap: 1rem`) | CSS variables (`gap: calc(var(--spacing) * 4)`) |
| `grid-cols-subgrid` | N/A | New (CSS subgrid) |
| `grid-rows-subgrid` | N/A | New (CSS subgrid) |
| `place-content-baseline` | N/A | New |
| `place-items-baseline` | N/A | New |
